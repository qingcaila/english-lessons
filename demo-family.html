👇 看下面每張卡片中間多一條淺橘色家族線,告訴你這個字的「原形」/「規則」/「家族關係」。
線上**可以點**(點了會跳到原形那個字)。

① 不規則動詞(過去式 / 過去分詞)

1 🚶 went ㄨㄝㄣㄊ ← go 過去式 w + ent 去
2 👀 saw ㄙㄛ ← see 過去式 s + aw 看見
3 🧠 thought ㄙㄛㄊ ← think 過去式 + p.p. th + ought (gh 不發音) 想 / 認為
4 🍴 eaten ㄧ-ㄊㄣ ← eat 過去分詞 eat + en 吃(p.p.)

② 拼字變化(規則動詞 / 複數 / 比較級)

5 🏃 running ㄖㄚ-ㄋㄧㄥ ← run +ing(雙 n) run + ning 跑步中
6 📚 studied ㄙㄉㄚ-ㄉㄧㄉ ← study +ed(y→i) stud + ied 學習(過去)
7 👶 babies ㄅㄟ-ㄅㄧㄗ ← baby 複數 y→ies ba + bies 嬰兒(複)
8 📏 bigger ㄅㄧ-ㄍㄜ ← big +er 比較級 雙 g big + ger 更大的
9 🚶‍♂️ walked ㄨㄛㄎㄊ ← walk +ed 規則過去式 walk + ed 走(過去)

沒有家族線的字維持原樣

10 🏠 home ㄏㄡㄇ placeholder h + ome (silent-e o-e) 家
