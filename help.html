1. 這是什麼?

這是專為「想看英文影片、讀原文文章」打造的拆字唸 + 間隔複習學習系統。

核心理念

  • 📖 L4 才算數:不是「學過多少字」,是「真的會看懂多少字」
  • 🧩 拆字唸:看到陌生字會自己拆音節讀出來
  • 🔄 間隔複習:在快忘的時候複習(不是同一天念 100 遍)
  • 🎯 80% 規則:現有字沒到 80% L4 前,不狂加新字
📌 你的目標:看英文影片、讀原文文章。L4 達到 5,000-7,000 字就能讀 Medium 文章了。

5. 課程頁面操作

開始一輪後會看到 10 張卡片。每張卡片包含:

範例卡片(編號+圖示堆疊在左,單字佔中間)
1 home ㄏㄡㄇ 家 NEW ✅ ❌ 🏠 h + ome (silent-e o-e)

互動方式

  • 🔊 點卡片任何地方 → 唸整個英文字
  • 📏 點英文下面的底線音節 → 只唸該音節(例 "ome" 部分)
  • ✅ 會 → 字下次出現時間延後(間隔×2)
  • ❌ 不會 → 字下次很快又出現(間隔÷2)
  • 🆕 NEW 紅標 → 還沒互動過,點任一處消失
  • 🔊 ×3 → 顯示你點了幾次(點太多次=還不熟)

頂部控制列

  • ← → 回首頁
  • 🔊 → 全部依序唸完整輪的字
  • ⏹ → 停止播放
  • 速度下拉 → 慢 / 正常 / 較快 / 原速(預設慢)
  • 注音 按鈕 → 切換注音顯示(預設開)
  • 結束 → 橘色按鈕 → 隨時可按,立刻結束本輪不用標完所有字

📋 控制列下方 chip 顯示本輪資訊:R1 · 模式 · 🔁複習 · 🆕新字 · 累計 · L4

結束本輪

兩種方式觸發結束:

  • 標完全部 10 字的 ✅/❌ → 自動跳 summary
  • 隨時按 topbar 的 結束 → → 立刻跳 summary,沒標記的字會延到下一輪

結束時 summary

summary 會顯示:

  • ✅ 會幾個 / ❌ 不會幾個 / 正確率(只算已標的)
  • ⏭ 幾個沒標(會延到下一輪)
  • 累計 L4 字數

然後三件事可做(都不強制):

  1. 選 emoji 自評(可選,不選也行)
    emoji系統回應
    😎 太簡單建議升難度 / 試挑戰模式
    👍 剛剛好甜蜜帶,維持節奏
    😅 偏難連續 2 次觸發放慢對策
    😵 太難立刻降一級,重建信心
  2. 按 ▶ 下一輪 → 延用同設定立刻繼續
  3. 按 🏠 回首頁 → 換模式 / 暫停
💡 重點:**不選 emoji 也能走**,直接點下一輪 / 回首頁就行。emoji 純粹幫系統優化下一輪推薦。

7. L1-L5 是什麼?

每個字學到的「真會程度」分 5 層:

層意思
L1 純理解看到字 + 中文能對得起來
L2 會閱讀看到字會唸出來
L3 模擬應用能造句
L4 ⭐ 被動真實應用讀文章時反射理解,不用查
L5 主動真實應用口說 / 寫作能主動用

怎麼升到 L4?

  • 連續按「✅ 會」2 次 + 點唸次數 < 5 → 升一層
  • 每次間隔複習成功(在快忘時答對)→ 進長期記憶
  • 約 8-10 次成功複習 → 穩定停在 L4
🎯 字量目標(L4):Stage 0: 500-1K / Stage 1: 2-3K / Stage 2: 4-5K / Stage 3: 5-7K(讀 Medium)/ Stage 4: 8-10K+(原文書)

8. Stage 切換

首頁頂部下拉選單可以切換 5 個 Stage:

Stage內容
0 超級基礎phonics 地基、500-1K 字
1 基礎過去式 / 片語動詞、2-3K 字
2 中下News in Levels、4-5K 字
3 中Medium 文章、5-7K 字
4 中上原文書 / 學術、8-10K+ 字
⚠️ 重要:切換 Stage 永遠不會洗掉你的學習進度。已達 L4 的字永遠是 L4。切換只是換「看哪個階段的字」。

Stage 影響哪些東西

  • ✅ 「目前位置」字量目標範圍(Stage 0: 500-1K / Stage 4: 8-10K)
  • ✅ 字量進度條%(L4 數 ÷ Stage 最低目標)
  • ✅ 新一輪抽字池(只抽 ≤ 當前 Stage 的字,見下方「字池模式」)
  • ✅ 挑戰模式抽什麼字(stage = currentStage + 1)
  • ✅ 主題 tile 軟暗示(超過 Stage 的主題會虛線淡化)

🎯 字池模式(累積制 vs 獨佔制)

Stage 下拉旁邊有個 checkbox 「只練本階段」,切換兩種模式:

模式抽字池舉例(Stage 2)
📌 累積制(預設)stage ≤ currentStageStage 0 + 1 + 2 字都抽得到(5,857 字)
📌 獨佔制(勾選)stage === currentStage只抽 Stage 2 的字(3,225 字)

下方提示文字會即時顯示目前模式 + 可抽字數。

💡 怎麼選?
• 想複習舊字+學新字(自然累積)→ 累積制
• 只想專注本階段挑戰(像課表)→ 獨佔制
該複習(due)的字不受影響,到期一律推。Stage 過濾只影響「抽新字」。

🚧 主題軟暗示

選「🎯 指定主題」時,超過你 Stage 的主題會虛線邊框+淡化。點下去仍可進(會跳 confirm「這是 Stage X 主題,你目前 Stage Y,確定要練嗎?」)。

注意:主題雖然可越級進,但 Stage 字池過濾仍生效,可能抽不到字。要練進階主題建議先升 Stage。

所有 73 個主題都可點,切換 Stage 不會鎖主題,只影響抽字池。

9. 小技巧

  • 💡 不確定某個字怎麼唸? 點底線音節單獨唸,聽清楚再點整字
  • 💡 同一個字點了 5 次以上發音? 系統會擋你升級,代表還不熟,別急著按「會」
  • 💡 累了想休息? 按 topbar 的 結束 → 立刻退出(沒標的字延到下一輪),或直接關掉分頁也行,已標記的會記住
  • 💡 連續 2 次 😅? 系統會建議下輪減新字、多複習
  • 💡 想衝刺? 把「新/複習比」拉到 70/30 或 100/0
  • 💡 換手機? 進階 → 匯出 JSON → 另一台匯入
  • 💡 很累很煩複習? 直接去看英文影片、讀英文文章,字在自然語境中重逢效果更好
  • 💡 覺得字池太雜? 在「目前位置」右側勾「只練本階段」 → 切到獨佔制,只練當前 Stage 的字(像課表)。詳見 §8
  • 💡 看到主題 tile 是虛線淡的? 那是超過你 Stage 的主題,點下去會問你「確定要練嗎?」,確定後仍可進(但字池被 Stage 過濾,可能抽不到字 → 要嘛先升 Stage,要嘛取消勾「只練本階段」)

10. 遇到問題?

聲音不會唸 / 唸得很怪

  • 確認手機沒靜音
  • iOS 設定 → 輔助使用 → 朗讀內容 → 聲音 → 英文 → 下載「增強版」女聲
  • 第一次點 🔊 可能要等 1 秒載入聲音

頁面壞掉 / 卡住

  • 下拉重整頁面(Safari 從頂部下拉)
  • 關掉分頁重開
  • 仍不行 → 進階「重置進度」(會清掉 localStorage,**進度會消失**)

字量 / 主題狀況

  • 目前 10,002 字,73 個主題,涵蓋大考中心 7000 字表 + 衍生(idioms / phrasal verbs / AWL 學術詞 / 抽象名詞)
  • 所有主題都可點,無灰色鎖定
  • 各主題字數不均(例:📚 學術詞 747、🐶 動物 201、🍴 廚房用具 58)— 自然分布,不影響使用

進度不見了

  • 進度存在這台手機 / 瀏覽器的 localStorage
  • 清瀏覽器資料 / 換瀏覽器 / 換手機 → 進度會消失
  • 解法:**定期匯出 JSON 備份**(進階 → 匯出進度 JSON)
