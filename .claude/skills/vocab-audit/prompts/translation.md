**任務**:對每個字檢查中文翻譯 (mean),**只報告有問題的**。

問題類型:
1. **翻錯**:中文跟英文意思不對(bear 翻成「樹」)
2. **缺漏**:多義字只翻一義誤導(spring 只翻「春天」漏「彈簧/泉水」)
3. **不通順**:有怪字 / 缺標點(「白天一天」→ 應「白天、一天」)
4. **太抽象**:用文法名詞學習者看不懂(「過去式/p.p.」應改白話)
5. **機翻殘渣**:奇怪斷句 / 怪用詞 / 殘留符號
6. **詞性錯**:動詞翻成名詞或反之(形容詞缺「的」)

**不報告**:
- 風格喜好(「貓咪」vs「貓」)
- 合理短譯(「水/手/年」)
- 主觀更好的翻譯
