📝 各 voice 來源說明

  • 本機 (Web Speech API):你目前 lesson.html 用的就是這個 — 完全免費可商用,品質看裝置(iPhone 上的 Siri 系列最好;Android Chrome / Windows 普通)。直接套用 = 0 工作量,但跨裝置音質不一致。
  • AWS Polly Neural:Readle 級的雲端 Neural TTS。這頁用的是 StreamElements 的公開測試 endpoint(Twitch 主播在用),不能正式商用但可以聽音質。正式版要你自己申請 AWS 帳號 + 用 API key + 預生成 mp3 存 CDN(免費額度 12 個月內每月 100 萬字元,夠你 10K 字跑很多次)。
  • Google Translate TTS:Google 翻譯網站用的端點。同樣是公開但非授權商用。某些瀏覽器可能因 CORS 播不出來,聽得到就好。

建議流程:先聽哪種 Polly voice 你最喜歡 → 我幫你寫腳本用 AWS 免費額度一次生 10K mp3 → 傳 Cloudflare R2 → 改 lesson.html 用 Polly 音檔,fail 時 fallback 到 Web Speech。
