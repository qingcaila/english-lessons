## 選字偏差(誠實揭露)

### 偏向「input 用」(刻意,符合 profile)
- 日常 / 食物 / 動物 / 影視 / 旅遊 — 比考試重視
- 口語感嘆詞(wow / hey / oops / congrats)— 考試不考
- 外來食物詞(sushi / taco / sake / 熱帶水果)— 考試極少出
- 圖像感強的具象名詞 — 比抽象詞優先

### 考試會考但暫時不夠的(會在後段批次補)
- **AWL 學術詞 ~570 字** — D3-D5 規劃中
- **商業 / 辦公室英文** — C 階段(C3-C4)
- **抽象思辨詞**(philosophy / politics / law)— C-D 階段
- **片語動詞** — B6 / D7 規劃中
- **連接副詞密集化** — sight-words 中已部分加入

## 結論

1. **路徑沒偏**:10K 達成後**自然覆蓋上述所有檢定的字量門檻**,不必專門為考試補字。
2. **選字偏差合理**:目標是 input(認得就好,L4),不是 output(主動使用,L5)。考試重 L5,所以表面數字會略低於同字量考生。
3. **若未來想考**:D9 收尾後加做:
   - 驗證 AWL 是否完整
   - 補考試型慣用語(雅思 academic linking、TOEIC 商業情境)
   - 做題型訓練(不是字量問題)

## 不變動原則

- 目前批次表 B6→D9 不調整
- 不為了考試覆蓋率刻意提前某些字
- D9 收尾後再評估是否需要針對特定考試補強
